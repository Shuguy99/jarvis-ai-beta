## 1. Быстрый старт

### Установка

```bash
git clone <url-репозитория>
cd jarvis-source
npm install
npx prisma db push
npm run dev
```

После запуска откройте `http://localhost:3000` в браузере.

### Настройка ИИ-провайдера

JARVIS может работать с несколькими провайдерами. Настройте нужный в файле `.env` (скопируйте из `.env.example`):

**Ollama (локальный ИИ, бесплатно):**

1. Установите Ollama с [ollama.com](https://ollama.com)
2. Скачайте модель: `ollama pull llama3.1`
3. Ollama автоматически запускается на `localhost:11434`

```
OLLAMA_BASE_URL=http://localhost:11434/v1
OLLAMA_MODEL=llama3.1
```

**OpenAI:**

```
OPENAI_API_KEY=sk-ваш-ключ
```

**Anthropic (Claude):**

```
ANTHROPIC_API_KEY=sk-ant-ваш-ключ
```

**Google Gemini:**

```
GEMINI_API_KEY=ваш-ключ
```

**OpenRouter (универсальный):**

```
OPENROUTER_API_KEY=sk-or-ваш-ключ
```

Провайдер выбирается в **Настройках** (кнопка ⚙ в шапке).

## 2. Интерфейс

### Шапка (Header)

| Кнопка | Описание |
|--------|----------|
| **Commands** | Открывает командную палитру для быстрого поиска действий |
| **Notes** | Открывает/закрывает панель заметок |
| **Wake** | Включает/выключает распознавание фразы «Hey Jarvis» |
| **Voice / Muted** | Включает/выключает авто-озвучку ответов JARVIS |
| **DnD** | Режим перетаскивания виджетов (Drag & Drop) |
| **Alerts** | Центр уведомлений |
| **⚙ (Settings)** | Открывает панель настроек |
| **SUIT (Mark 1 / 42 / 50)** | Переключатель тем оформления |
| **Полноэкранная кнопка** | Разворачивает на весь экран |
| **Часы** | Текущее время |

### Левая панель

- **Системный монитор** — загрузка ЦП, ОЗУ, диска в реальном времени
- **Лента активности** — лог действий и событий
- **Список диалогов** — история чатов с возможностью переключения
- **Виджеты** — дополнительная информация (зависит от выбранной раскладки)

### Центральная область

- **Arc Reactor** — анимированный индикатор, отражающий состояние JARVIS (ожидание, обработка, ошибка)
- **Панель чата** — основная область общения с ИИ
- **Быстрые команды** — кнопки для частых действий (создать картинку, анализ экрана, экспорт и т.д.)

### Правая панель

- **Сетка возможностей** — визуальная карта всех функций JARVIS
- **Погода** — текущая погода
- **Мировые часы** — время в разных часовых поясах
- **Музыкальный плеер** — управление аудио
- И другие виджеты (зависят от выбранной раскладки)

### Футер (Статус-бар)

Показывает текущее состояние системы: провайдер, модель, статус подключения.

### Quick Actions Bar

Панель быстрых действий внизу экрана — кнопки для часто используемых функций. Сворачивается по клику на стрелку.

## 3. Чат и голос

### Ввод текста

Напишите сообщение в поле ввода внизу чата и нажмите **Enter** или кнопку отправки.

### Голосовой ввод

1. Нажмите кнопку **микрофона** ⏺ рядом с полем ввода
2. Разрешите доступ к микрофону в браузере
3. Говорите — текст появится в поле ввода автоматически
4. Нажмите **Enter** для отправки

### Авто-озвучка

Когда включена (кнопка **Voice** в шапке горит), JARVIS автоматически читает свои ответы вслух. У каждого ответа также есть кнопка повтора для повторного озвучивания.

### Wake Word — «Hey Jarvis»

1. Включите Wake Word кнопкой **Wake** в шапке (загорается индикатор пульса)
2. Скажите «**Hey Jarvis**» — появится оверлей и звуковой сигнал
3. Продиктуйте команду или вопрос — JARVIS выполнит

### Голосовые команды (NLP)

JARVIS понимает естественные фразы на русском и английском:

- «**Открой заметки**» / «**Open notes**» — откроет виджет
- «**Таймер 10 минут**» — запустит таймер
- «**Какая погода?**» — покажет прогноз
- «**Который час?**» — скажет время
- «**Смени тему**» — переключит тему оформления
- «**На весь экран**» — включит полноэкранный режим
- «**Новый чат**» — начнёт новый диалог
- «**Найди...**» — выполнит веб-поиск
- «**Помодоро**» — запустит режим фокусировки
- «**Выключи звук**» / «**Включи озвучку**» — управляет авто-озвучкой

## 6. Загрузка документов (RAG)

Вы можете загрузить текстовые файлы, чтобы JARVIS мог искать по ним и использовать контекст в ответах.

### Поддерживаемые форматы

`.txt`, `.md`, `.json`, `.csv`, `.xml`, `.py`, `.ts`, `.js`

### Как загрузить

1. Откройте **RAG Document Manager** (через командную палитру `Ctrl+K` → «RAG» или из панели виджетов)
2. Нажмите кнопку **Upload** или перетащите файл в область загрузки
3. Файл будет разбит на фрагменты и сохранён в базу данных

### Поиск по документам

1. Введите поисковый запрос в поле **Search**
2. Результаты покажут релевантные фрагменты с указанием файла и оценки совпадения

### Удаление документов

1. В списке загруженных документов нажмите иконку **корзины** 🗑 рядом с нужным файлом

## 7. ИИ-Агент

**Агентный режим** — это автономный режим, в котором JARVIS сам планирует и выполняет последовательность действий для решения задачи.

### Как использовать

1. Откройте панель **Agent** (кнопка в интерфейсе или через командную палитру)
2. Опишите задачу текстом
3. JARVIS составит план из шагов и начнёт выполнение
4. Каждый шаг отображается с прогрессом и результатом

### Ограничения

- Максимум **5 вызовов инструментов** на одну задачу
- Агент может искать в сети, проверять систему, анализировать данные и т.д.
- Если задача сложная, разбейте её на несколько запросов

Некоторые плагины имеют дополнительные настройки (интервал обновления, макс. символов и т.д.). Нажмите на стрелку рядом с плагином, чтобы раскрыть настройки.

## 9. Темы

JARVIS включает 3 встроенные темы в стиле костюмов Железного Человека:

| Тема | Описание |
|------|----------|
| **MARK 1** | Классический голубой циан — оригинальный интерфейс JARVIS |
| **MARK 42** | Золотой/янтарный — стиль костюма Mark 42 |
| **MARK 50** | Красный/огненный — стиль костюма Mark 50 (Нанотех) |

### Переключение

1. В шапке найдите блок **SUIT** с тремя кнопками
2. Нажмите на нужную тему — переключение произойдёт мгновенно
3. Выбор сохраняется в `localStorage` и сохраняется между сессиями

Также можно переключить тему голосовой командой: «**Смени тему**» или «**Change theme**».

## 10. Решение проблем

### «Ollama not running» / Ошибка подключения к Ollama

1. Убедитесь, что Ollama установлен: `ollama --version`
2. Запустите Ollama: `ollama serve` (или перезапустите приложение Ollama)
3. Проверьте, что модель скачана: `ollama list`
4. Если модели нет: `ollama pull llama3.1`
5. Убедитесь, что `OLLAMA_BASE_URL` в `.env` указывает на `http://localhost:11434/v1`

### Как сбросить настройки

Все настройки JARVIS хранятся в `localStorage` браузера:

1. Откройте DevTools браузера (`F12`)
2. Перейдите на вкладку **Application** → **Local Storage**
3. Найдите записи, начинающиеся с `jarvis-`
4. Удалите нужные записи или очистите всё

### Совместимость браузеров

Для корректной работы рекомендуется:

- **Chrome / Edge** 90+ — полная поддержка всех функций
- **Firefox** 90+ — полная поддержка
- **Safari** 16+ — поддерживается, но могут быть ограничения с Web Audio API

Для голосового ввода и Wake Word необходим браузер с поддержкой **Web Speech API** (Chrome, Edge).

### Прочие проблемы

- **Микрофон не работает** — проверьте разрешения браузера (значок замка 🔒 в адресной строке)
- **Чёрный экран / белый экран** — очистите кэш браузера (`Ctrl+Shift+Delete`)
- **Медленная работа** — проверьте загрузку ЦП, закройте лишние вкладки
